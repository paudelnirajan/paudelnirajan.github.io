About Me

I’m Nirajan Paudel, a Master’s student in Computer Science at the University of Colorado Boulder, with a Bachelor’s in Electronics, Communication & Information Engineering from Tribhuwan University. I’m passionate about AI and machine learning, actively exploring multiple fields like NLP, Computer Vision, Reinforcement learning to build innovative solutions. My experience includes working on AI-driven chatbots during my internship at PalmMind Technology and teaching programming as a Teaching Assistant at Pashchimanchal Campus. Through personal projects like conversational AI applications, I’m learning to create efficient, context-aware systems using Python, C++, and frameworks like TensorFlow. I’m excited to continue growing and contributing to impactful AI projects.

I wonder, will humans be able to preserve their culture and language in AI models that can persists longer than human themselves?

Projects

AI-Powered Question Assistant (IOE-GPT)

Developed an AI-powered system for IOE students, leveraging LangChain and Groq LLM to retrieve and answer past questions with semantic and metadata-based search.

GitHub Repository

Audio RAG Assistant

A web application that transcribes audio using OpenAI’s Whisper and enables intelligent question-answering with Retrieval-Augmented Generation (RAG).

GitHub Repository

digitalME-Personal Assistant

A personal AI-powered chatbot designed for natural conversations and assisting with tasks like document retrieval and database queries.

GitHub Repository

PDF Chat Assistant

An AI-powered chatbot that allows users to upload PDF documents and interact with them through natural conversations using the Mistral-7B-Instruct model.

GitHub Repository

GitHub Repository

Nepali Image Captioning

Developed a model for generating paragraph-length Nepali captions using Transformer architecture with Inception V3 feature extraction. Compared Transformer against LSTM with ResNet152, optimizing hyperparameters.
